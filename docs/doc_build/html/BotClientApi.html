bot.telegram_bot¶

This is a core module for Telegram bot version.

Adjusts bot replies to user command/messages.

bot.telegram_bot.run_channel(config: Dict[str, Union[str, Dict[str, str]]]) → None¶

Function to activate Telegram bot custom behaviour in a channel It runs endless while cycle, to terminate - press Ctrl+C

Parameters

config – dictionary with parameters related to server and bot client

For details check README.md file in repo root :return: None

Usage:
>>> import json
>>> from bot import telegram_bot
>>> with open('./utils/my_config.json', 'r') as rf:
...     config = json.load(rf)
...
>>> import logging
>>> logging.getLogger('lexibot').setLevel(logging.DEBUG) # or logging.INFO
>>> telegram_bot.run_channel(config)
telegram_bot.py:198 Starting up the <bot_username> bot custom behaviour...
telegram_bot.py:108 No pending updates
...
^Ctelegram_bot.py:219 Finishing custom bot behaviour.
bot.telegram_bot.run_pm(bot_token: str) → None¶

Function to activate Telegram bot custom behaviour in private messages. It is considered as test function to verify whether telegram responds properly.

Parameters

bot_token (str) – Telegram bot token

Returns

None

Usage:

>>> token = "<your_token>"

bot.telegram_bot.send_regular_report(bot: telegram.bot.Bot, channel_id: str, server_ip: str = '0.0.0.0:80') → None¶

Send regular report about user clicks on word links

Parameters

  • bot – telegeram.Bot instance created via unique Telegram bot token

  • channel_id – id field to initialize telegram.Chat instance. For private channels telegram.Chat can only be obtained by calling telegram.User.get_chat(id=<id>) or telegram.Bot.get_chat(id=<id>) for the users/bots - participants of the channel

  • server_ip – server ip address + port. Bot will communicate with server using this info.

bot.telegram_bot.send_word_links(bot: telegram.bot.Bot, channel_id: str, message: telegram.message.Message, server_ip: str = '0.0.0.0:80') → None¶

Reply to channel message by providing links to online dictionary For every unique japanese word link is generated and sent to chat

Parameters

  • bot – telegeram.Bot instance created via unique Telegram bot token

  • channel_id – id field to initialize telegram.Chat instance. For private channels telegram.Chat can only be obtained by calling telegram.User.get_chat(id=<id>) or telegram.Bot.get_chat(id=<id>) for the users/bots - participants of the channel

  • message – telegram.Message instance which represents message in telegram channel

  • server_ip – server ip address + port. Bot will communicate with server using this info.

bot.parser¶

This is a module with text parsing methods

bot.parser.extract_japanese_words(message: str) → List[str]¶

Each non-japanese word is skipped. Japanese words are extracted.

Japanese text has no whitespaces, that’s why tokenization is required

bot.parser.is_japanese_char(symbol: str) → bool¶

Test whether character can be found in japanese text according to Unicode code
