Description¶

server.application¶

This module contains the server itself and handlers for it’s endpoints.

server.application._initialized()¶

Return True if all parameters of the application were set.

server.application.all_exception_handler(error)¶

Handle all exceptions.

server.application.click(link_id: int)¶

Handle short-link click.

server.application.create()¶

Create new short-link.

Takes JSON object as an input, which should contain chat id as ‘chat_id’, word as ‘word’ and bot token hash as ‘token_hash’.

Returns the ID of newly created link.

server.application.get_top(chat_id: str)¶

Return pairs of (word, number of clicks) for top 5 clicked words for chat with ‘chat_id’.

Takes optional parameter ‘date’. The result will be filtered for the time period, which includes date. If ‘date’ parameter is not provided it is defaulted to current date in UTC.

server.application.initialize_app(conn, bot_token_hash) → flask.app.Flask¶

Return Flask application initialized with DB connection.

server.model¶

This module contains functions to work with link DB.

server.model._full_link(provider, word)¶

Return a website link for dictionary provider and word.

server.model.create(conn, chat_id: str, word: str, provider: str = 'jisho.org/search/') → str¶

Insert new word in chat with id chat_id, returns id of the inserted word.

server.model.get_words_with_clicks(conn, chat_id: str, current_date: datetime.date)¶

Return top 5 most clicked words as pairs (word, clicks) for chat with chat_id and with start-end interval that includes current_date.

server.model.handle_click(conn, link_id: int) → str¶

Increase click count for word with id link_id and returns full dictionary link to the word.
